---

Sure includes an AI assistant sidebar on desktop. If you do not want to use it yet, you can hide it and bring it back later.

## Hide or show the AI sidebar on desktop

On desktop, use the **panel-right** button in the top bar to toggle the right-hand AI sidebar.

- Click it once to collapse the AI sidebar
- Click it again to reopen the AI sidebar
- Sure saves this preference to your account

This is useful if you want more room for the main app content or simply do not want to use the assistant yet.

## When the sidebar is hidden by default

Sure can also hide the AI sidebar by default for some people:

- New `member` and `guest` users start with the AI sidebar hidden when AI is not available
- `admin` users still see it by default so they can finish AI setup or troubleshooting

If AI becomes available later, you can reopen the sidebar at any time with the same desktop toggle.
